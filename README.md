## Description
ReLeaf is a community-based web application that promotes environmental awareness while encouraging collaboration among individuals who care about sustainability.

The platform serves as an all-in-one space where users can explore eco-friendly practices, share knowledge, exchange items, and participate in community-driven activities.

---

## Objectives
- Promote sustainable living habits
- Provide an information hub for eco-friendly tips
- Encourage community interaction and collaboration
- Reduce waste through item swapping
- Ensure a safe and trustworthy platform for users

---

## Key Features
- 👤 User Registration & Profile Management  
- 📝 Blog Posting (energy saving, recycling, gardening tips)  
- 📅 Event Hosting & Participation  
- 🔄 Eco-product Swap Marketplace  
- 💬 User Communication (chat system)  
- 🛡️ Reporting & Verification System (anti-scam protection)  
- 🎮 Quiz & Gamification (user engagement)  
- 🛠️ Admin Dashboard (moderation & support ticket handling)  

---

## Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL

## ⚙️ Setup Note (XAMPP Requirement)

Please ensure that **XAMPP is installed and properly configured** on your system before running this project.

Make sure the following modules are started in the XAMPP Control Panel:
- Apache
- MySQL

The project will not run correctly if these services are not active.

Additionally, ensure the database `rwddgroup7` (`rwddgroup7.sql`) has been imported via phpMyAdmin before testing the system.

- NOTE: The dataset used in this project consists of **dummy data only** and is intended strictly for development, testing, and demonstration purposes.
